# Pog-16

<br>

## About

A 16 bit, fully custom architechture

__Hardware:__

* 16 bit data bus
* 64kb ram
* 5 general purpose registers
* 64kb stack

---

## Building

Included build is for windows, to manually build use:
* Emulator: `gcc source\emulator.c -o emulator.exe`
* Assembler `python setup.py build`
